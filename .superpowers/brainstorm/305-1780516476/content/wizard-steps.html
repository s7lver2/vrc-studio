¿Cómo organizamos los pasos del wizard?

El diálogo actual tiene todo en un scroll interminable. Aquí hay 3 formas de dividirlo.

1
📁 Archivo + Modo (Single / Multi)
2
🏪 Booth (opcional) → auto-rellena lo demás
3
✏️ Metadata (nombre, autor, thumbnail)
4
🏷️ Tags + Avatares
5
✅ Resumen + Importar

A — 5 pasos lineales

Flujo completo paso a paso. El paso 2 (Booth) rellena automáticamente el 3 y 4. Para Multi Avatar, el paso 2 incluye el mapeo de variantes.

1
📁 Archivo — arrastra o selecciona
2
🔍 Tipo detectado + Booth (todo junto)
3
🖼️ Info + Imagen (nombre, autor, thumbnail, tags)
4
✅ Confirmar e importar

B — 4 pasos, más compacto

El paso 1 es solo "elige el archivo" — simple y claro. El paso 2 agrupa detección automática y Booth. El paso 3 agrupa todo lo de metadata + imagen.

1
📁 Archivo + modo (drop zone grande)
Si Multi →
variantes aquí
Si Single →
skip directo
2
🏪 Booth + nombre + autor
3
✅ Confirmar e importar

C — 3 pasos (adaptativo)

El paso 1 es adaptativo: si es Multi, aparece el mapeo de variantes ahí mismo. Booth rellena el paso 2 automáticamente. Mínimo 3 clics para importar.
